# Sistema de Partículas - Tiro Parabólico

Proyecto para la asignatura de Física - Grado en Diseño y Desarrollo de Videojuegos

## Descripción
Sistema de partículas el cual se activa al pulsar la tecla espacio.

## Características
- Partículas con velocidad, ángulo y tiempo de vida variables
- Sistema configurable desde el inspector
- Generación de partículas al presionar ESPACIO
- Movimiento parabólico con gravedad

## Controles
- **ESPACIO**: Crear nueva explosión de partículas
